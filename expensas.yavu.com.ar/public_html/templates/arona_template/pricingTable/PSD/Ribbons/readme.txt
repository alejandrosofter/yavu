With ribbons.psd file you can create your own or modify existing column ribbons.
It's a photoshop CS3 file with smart layers.

Before you start please make sure you have Cabin font installed in your OS. This font is included in Font directory within this package.

1. Please open ribbons.psd file - you'll see two layer groups: Style1 and Style2.
2. Please expand for e.g. Style1 group by clicking on the blue arrow.
3. Now you'll see three layers, please follow Right Click / Edit Contents instruction - this action will open a new window.
4. Make your edits and close the active window (saving all changes).
5. It's done! Now save for web (ALT+SHIFT+CTRL+S), select PNG-24 with transparent option enabled.

Good luck! :-)
